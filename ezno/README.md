## How to use

Install with:

```sh
npm i --save @ganesha/ezno
```

Use from shell:

```sh
node --import @ganesha/ezno my-program.ts
```

Use from script:

```js
await import("@ganesha/ezno")
await import("./my-program.ts")
```
