## How to use

Install with:

```sh
npm i --save @hackbg/ganesha
```

Use from shell:

```sh
node --import @hackbg/ganesha my-program.ts
```

Use from script:

```js
await import("@hackbg/ganesha")
await import("./my-program.ts")
```
