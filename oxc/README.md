## How to use

Install with:

```sh
npm i --save @ganesha/oxc
```

Use from shell:

```sh
node --import @ganesha/oxc my-program.ts
```

Use from script:

```js
await import("@ganesha/oxc")
await import("./my-program.ts")
```
